# Smart To-Do List

A clean and responsive To-Do web app with filters and local storage.

## 🔗 Live Demo
https://afraanees92.github.io/smart-todo-list

## ✨ Features
- Add, delete, and mark tasks as complete
- Filters: All / Active / Completed
- Persistent storage using localStorage
- Simple, responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 How to Run Locally
1. Clone the repo
2. Open index.html in your browser

## 📌 Future Improvements
- Dark mode
- Drag & drop tasks
- Due dates & reminders
- Mobile-first redesign
